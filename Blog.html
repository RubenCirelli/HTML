Introduction

what is a computer?

A computer is an electronic device that processes information.

  1. Input A computer takes in raw information or data. This could be through devices like a keyboard, mouse, or even a touch screen.
  2. StorageThe computer stores this information until it’s ready to work on it. This is done in the computer’s memory.
  3. ProcessingThe computer then works on this information, performing calculations and operations based on instructions provided by a software or hardware program. This is done by the computer’s processor.
  4. OutputFinally, the computer produces results, which could be displayed on a screen, printed on paper, saved to a disk, sent to another computer, and so on.
